# switch

```java
public class Mathx {
  public static int fibonacci(int n) {
    switch (n) {
      case 0:
        return 0;
      case 1:
        return 1;
      default:
        return fibonacci(n - 1) + fibonacci(n - 2);
    }
  }
```

- return 0; 지우면? (fall-through 조심, default 빼먹을 때 조심)
- return 대신에 모두 break를 쓰면?
- switch (n) .. 에서 n에 올 수 있는 type은?
